# GPS Tracking Simulation Web

A GPS device tracking simulation built with **Vite + React + TypeScript + Tailwind CSS**.

## Features

- **Device List Dashboard** — Browse all registered GPS devices
- **Search** — Filter devices by name in real time
- **Status Filter** — Toggle between All / Active / Inactive devices
- **Device Cards** — Each card shows name, status, last-seen time, battery level and address
- **View / Edit actions** — Quick-access buttons on every card
- **Empty state** — Friendly message when no devices match the current filters
- **Mock API** — Simulated API layer with realistic delay (no real backend required)

## Getting Started

```bash
npm install
npm run dev
```

Open <http://localhost:5173> in your browser.
